# 血压分级参考标准

## 中国高血压防治指南（2024）

| 分级 | 收缩压 (mmHg) | 舒张压 (mmHg) |
|------|--------------|--------------|
| 正常 | < 120 | < 80 |
| 正常高值 | 120-139 | 80-89 |
| 高血压 1 级 | 140-159 | 90-99 |
| 高血压 2 级 | 160-179 | 100-109 |
| 高血压 3 级 | ≥ 180 | ≥ 110 |

## 异常阈值

- 收缩压 ≥ 140 或 ≤ 90 → 异常
- 舒张压 ≥ 90 或 ≤ 60 → 异常
- 脉压差 > 60 → 需关注动脉硬化风险
