**Domains:**
3. Patient Manager Assignments (`patient_manager_assignments`)

### Task 1: Patient Manager Assignments Entity & Controller (Java & NestJS)

**Goal:** Provide endpoints to assign, list, and unassign managers for patients based on soft RLS boundaries.

- [ ] **Step 1: NestJS Entity & Module**
  - Create `PatientManagerAssignmentEntity` mapping to `patient_manager_assignments` table. Provide columns: `id`, `tenant_id`, `org_id`, `patient_id`, `manager_user_id`, `assignment_type` (main/assistant), `created_at`.
  - Create Service and Controller endpoints taking `IdentityPayload` and filtering by `allowedOrgIds`.
  
- [ ] **Step 2: Java Entity & Repository**
  - Create `PatientManagerAssignmentEntity.java` with JPA decorators.
  - Create `PatientManagerAssignmentRepository.java` extending `JpaRepository` with custom contextual queries.
  - Create Controller and Service layers.
